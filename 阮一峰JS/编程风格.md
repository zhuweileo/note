## 缩进
使用空格缩进，可以保持代码在不同编辑器中的统一

## 区块括号
```
return {
  key : value
};
```

## 虽然js允许省略行尾分号，但是我们尽量还是不要省略
```
return
{ first: 'Jane' };

// 解释成
return;
{ first: 'Jane' };
```

## 避免使用全局变量
用大写字母表示全局变量名
